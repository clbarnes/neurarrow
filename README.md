# neurarrow

Specifications for neuroscience data using the [Apache Arrow](https://arrow.apache.org/) data model,
and models compatible with it (e.g. [Apache Parquet](https://parquet.apache.org/)).

Development happens [on github](https://github.com/clbarnes/neurarrow),
and the rendered specification is at <https://neurarrow.readthedocs.io>.

## This repository

This repository uses [mdBook](https://rust-lang.github.io/mdBook/) to produce a web page for the schema.
Install it and run `mdbook serve`.

## Contribution

Pull requests should use [`dev` branch](https://github.com/clbarnes/neurarrow/tree/dev) as a head.
