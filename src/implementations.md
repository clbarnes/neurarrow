# Implementations

Software implementing neurarrow should be listed here.

## Software libraries

- [neurarrow.py](https://github.com/clbarnes/neurarrow/blob/main/examples/python/neurarrow.py)
  - simple python/ pyarrow validator for neurarrow data
- [swc2na](https://github.com/clbarnes/swc2na)
  - rust crate for reading SWC files into neurarrow skeletons
  - uses the [`net.clbarnes.swc` extension](./extensions.md#known-extensions)

## User-facing tools

- [swc2na](https://github.com/clbarnes/swc2na)
  - CLI for converting SWC files into neurarrow feather and parquet files
  - uses the [`net.clbarnes.swc` extension](./extensions.md#known-extensions)
